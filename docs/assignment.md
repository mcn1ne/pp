# 슈퍼센트 AI 기반 제품 기획 과제

## 주제: 파트너 크리에이터 평가 자동화 도구

---

## ① 문제 정의

### 해결하려는 문제

슈퍼센트는 게임 홍보를 위해 YouTube 파트너 크리에이터를 모집하고 있다. 현재 마케팅 팀은 크리에이터 지원자를 **수작업으로 평가**하고 있으며, 이 과정에서 다음과 같은 문제가 발생한다:

- 크리에이터 1명당 평가에 **20~30분** 소요 (채널 방문, 영상 검토, 댓글 확인 등)
- 구독자 수, 조회수, 성장률 등을 **수동으로 계산**해야 함
- 댓글을 하나씩 읽어 시청자 반응을 파악하는 것이 **비효율적**
- 평가자마다 판단 기준이 달라 **일관성이 부족**
- 대량의 지원자를 빠르게 스크리닝하기 어려움

### 대상 사용자

- **게임 마케터** (슈퍼센트 파트너십/인플루언서 마케팅 담당자)
- 크리에이터 지원서를 검토하고, 파트너십 확대/축소를 결정하는 의사결정자

### 현재 업무 방식

1. 크리에이터가 지원서를 제출 (YouTube 채널 URL 포함)
2. 마케터가 YouTube 채널에 직접 방문
3. 구독자 수 확인 (500명 이상 기준)
4. 최근 업로드 영상 3~5개를 하나씩 클릭하여 조회수/좋아요 확인
5. 댓글을 스크롤하며 시청자 반응 파악
6. 채널의 성장 추세를 감으로 판단
7. 스프레드시트에 수동 기록 후 팀 내 논의

### 주요 불편 사항

| 불편 사항 | 설명 |
|-----------|------|
| 시간 소모 | 크리에이터 1명당 20~30분, 월 50명+ 지원 시 주당 10시간 이상 소요 |
| 댓글 분석 한계 | 수백 개 댓글을 사람이 읽고 감성을 요약하는 것은 사실상 불가능 |
| 기준 비일관 | 담당자별 주관적 판단으로 동일 크리에이터에 대해 다른 결론 도출 가능 |
| 성장률 계산 어려움 | 과거 영상과 최근 영상의 성과를 비교하려면 별도 정리 필요 |
| 데이터 기반 부족 | 정량 데이터 없이 직감에 의존한 파트너십 의사결정 |

## ② 해결 방법

### 기획하는 AI 제품: **파트너 크리에이터 평가 대시보드**

YouTube 채널 URL을 입력하면 **자동으로 채널/영상 데이터를 수집**하고, **AI가 댓글을 감성분석**하여, **종합 점수와 파트너십 추천 결과**를 대시보드로 제공하는 웹 도구.

### 핵심 기능

1. **채널 자동 분석**: YouTube API로 구독자 수, 총 조회수, 영상 수, 가입일 등 자동 수집
2. **영상 성과 분석**: 최근 영상의 조회수 추이, 좋아요/댓글 비율, 업로드 빈도 계산
3. **AI 댓글 감성분석**: Gemini API로 댓글의 긍정/부정/중립 비율 분석, 핵심 테마 추출
4. **종합 점수 산출**: 5개 항목(구독자, 참여도, 성장률, 감성, 일관성)을 가중 평균으로 계산
5. **파트너십 추천**: PASS / REVIEW / FAIL 자동 판정 + AI가 생성한 한국어 평가 요약

### 사용자 이용 흐름 (User Flow)

```
[마케터] YouTube 채널 URL 입력
    ↓
[시스템] YouTube API로 채널/영상/댓글 데이터 수집
    ↓
[시스템] Gemini API로 댓글 감성분석 수행
    ↓
[시스템] 5개 항목 종합 점수 계산
    ↓
[시스템] AI가 한국어 평가 요약문 생성
    ↓
[마케터] 대시보드에서 결과 확인
    ├── 채널 개요 (구독자, 조회수, 영상 수)
    ├── 차트 (조회수 추이, 참여도, 감성 분포)
    ├── 댓글 분석 (핵심 테마, 주목할 댓글)
    ├── 종합 점수 (항목별 + 총점)
    └── 파트너십 추천 (PASS/REVIEW/FAIL + AI 요약)
```

## ③ 사용할 AI 도구

### Google Gemini API (gemini-2.0-flash)

| 항목 | 설명 |
|------|------|
| 역할 | 댓글 감성분석 + 크리에이터 평가 요약 생성 |
| 선택 이유 | 한국어 처리 성능 우수, JSON 구조화 출력 지원, 비용 효율적 |
| 입력 | YouTube 댓글 텍스트 (한국어/영어 혼합) |
| 출력 | 감성 분류(긍정/부정/중립), 핵심 테마, 주목할 댓글, 종합 평가 요약문 |

### YouTube Data API v3

| 항목 | 설명 |
|------|------|
| 역할 | 채널/영상/댓글 데이터 수집 |
| 활용 | channels.list, search.list, videos.list, commentThreads.list |
| 일일 할당량 | 10,000 유닛 (분석 1회당 약 111 유닛, 일 약 90회 분석 가능) |

## ④ 프로토타이핑 계획

### 프로토타입 목적

- 마케터가 **실제 YouTube 채널 URL**을 입력하여 분석 결과를 확인할 수 있는 **동작하는 웹 프로토타입** 제작
- AI 기반 댓글 감성분석과 종합 점수 산출이 실제로 유용한지 검증

### 구현 범위

- [x] YouTube 채널 URL 입력 및 파싱
- [x] YouTube API로 채널/영상/댓글 데이터 수집
- [x] Gemini API로 댓글 감성분석
- [x] 5개 항목 종합 점수 계산
- [x] 대시보드 UI (차트 포함)
- [x] PASS/REVIEW/FAIL 추천 + AI 요약

### 사용할 도구

| 구분 | 도구 |
|------|------|
| 백엔드 | Python FastAPI |
| 프론트엔드 | HTML + Tailwind CSS + Chart.js (CDN, 빌드 불필요) |
| AI 분석 | Google Gemini API (google-genai SDK) |
| 데이터 수집 | YouTube Data API v3 (google-api-python-client) |
| 실행 환경 | uvicorn 로컬 서버 |

### 입력 데이터 및 출력 방식

**입력**: YouTube 채널 URL (예: `https://youtube.com/@channelname`)

**출력**: 웹 대시보드
- 채널 프로필 카드 (썸네일, 구독자, 조회수)
- 영상 성과 차트 (조회수 추이, 참여도, 업로드 빈도)
- 댓글 감성분석 결과 (도넛 차트 + 핵심 테마 태그 + 주목할 댓글)
- 종합 점수 카드 (항목별 바 차트 + 총점)
- 파트너십 추천 카드 (PASS/REVIEW/FAIL 배지 + AI 평가 요약문)

## ⑤ 프로토타이핑 방법

### 실제 프로토타입 제작 방식

1. **프로젝트 스캐폴딩**: FastAPI 기반 프로젝트 구조 생성
2. **백엔드 서비스 레이어 구현**:
   - `channel_resolver.py`: YouTube URL의 다양한 형식(/@handle, /channel/UCxxx 등)을 채널 ID로 변환
   - `youtube_service.py`: YouTube Data API v3로 채널 정보, 최근 영상, 댓글 수집
   - `gemini_service.py`: Gemini API에 댓글을 전송하여 감성분석 및 평가 요약 생성
   - `scoring_service.py`: 5개 항목(구독자, 참여도, 성장률, 감성, 일관성) 가중 점수 계산
3. **API 엔드포인트 구현**: `/api/v1/analyze/{channel_id}` - 전체 분석 파이프라인을 하나의 엔드포인트로 오케스트레이션
4. **프론트엔드 대시보드**: Tailwind CSS로 반응형 레이아웃, Chart.js로 데이터 시각화, Vanilla JS로 API 호출 및 렌더링
5. **통합 테스트**: 실제 한국 게임 유튜버 채널로 엔드투엔드 테스트

### 기술 아키텍처

```
[브라우저] ──HTTP──▶ [FastAPI 서버]
                        ├── YouTube Data API v3 ──▶ 채널/영상/댓글 데이터
                        ├── Gemini API ──▶ 댓글 감성분석 + 평가 요약
                        └── Scoring Engine ──▶ 종합 점수 계산
```

```
당신은 게임 회사의 마케팅 도구를 개발하는 시니어 풀스택 개발자입니다.

## 역할
슈퍼센트의 파트너 크리에이터 평가 자동화 웹 도구를 개발해주세요.

## 기능 요구사항
1. YouTube 채널 URL을 입력하면 자동으로 채널 데이터를 수집합니다
2. YouTube Data API v3로 구독자 수, 조회수, 최근 영상 목록, 댓글을 가져옵니다
3. Gemini API(gemini-2.0-flash)로 댓글을 감성분석합니다 (긍정/부정/중립)
4. 5개 항목으로 종합 점수를 계산합니다:
   - 구독자 점수 (20%): 500명 미만=0, 로그 스케일
   - 참여도 점수 (25%): (좋아요+댓글)/조회수 비율
   - 성장률 점수 (20%): 최근 vs 이전 영상 조회수 비교
   - 감성 점수 (20%): 긍정 댓글 비율
   - 일관성 점수 (15%): 업로드 빈도
5. PASS(60점+, 구독자500+) / REVIEW / FAIL 자동 판정
6. AI가 한국어로 평가 요약문을 생성합니다

## UI 요구사항
- 상단: 타이틀 "파트너 크리에이터 평가 도구"
- URL 입력 필드 + "분석 시작" 버튼 + 로딩 스피너
- 채널 개요 카드: 썸네일, 이름, 구독자, 조회수, 영상 수
- 2x2 차트 그리드: 조회수 추이(바), 참여도(도넛), 업로드 빈도(라인), 감성 분포(도넛)
- 댓글 분석 패널: 핵심 테마 태그, 주목할 댓글 목록
- 점수 카드: 항목별 수평 바 + 종합 점수
- 추천 카드: PASS/FAIL/REVIEW 배지(초록/빨강/노랑) + AI 요약문

## 기술 요구사항
- 백엔드: Python FastAPI
- 프론트엔드: HTML + Tailwind CSS CDN + Chart.js CDN + Vanilla JS
- API: YouTube Data API v3 (google-api-python-client), Gemini API (google-genai)
- 환경변수: .env 파일로 YOUTUBE_API_KEY, GEMINI_API_KEY 관리
- FastAPI에서 frontend/ 디렉토리를 정적 파일로 서빙
- 빌드 도구 없이 CDN만 사용
```
